## r/Angular (Day 4) — value-first, grade as evidence

**Title:** `How much of your app are you handing to code you didn't write? (supply-chain trust for Angular agent frameworks)`

**Body:**
> If you're building an agent feature into an Angular app, the framework underneath it runs models, calls tools, and reaches for credentials. So its supply chain matters more than most dependencies you pull in.
>
> I had Threadplane (our Angular agent framework) independently evaluated by HVTracker. It came back Grade A — 82.8/100, #7 of 75 agent frameworks, and the only Angular one on the board, ranked above CrewAI and LangChain on trust. What earned it: OSSF Scorecard 7.7/10, MIT license, signed commits, verifiable npm provenance.
>
> Not posting this as a trophy — it's *not* a safety endorsement, and our weakest signal is adoption. Posting it because "can I verify this framework's supply chain" is a question more of us should ask before we install something an agent will run.
