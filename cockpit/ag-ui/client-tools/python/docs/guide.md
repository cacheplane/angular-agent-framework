# Client Tools with AG-UI and Angular

<Summary>
Declare tools in the browser that the model calls and the browser executes.
A `function` returns a value, a `view` renders inline and auto-acknowledges,
and an `ask` is an interactive component whose emitted value resumes the run.
The catalog ships to the model over AG-UI; the LangGraph backend binds the
client stubs with no server implementation and ends the turn so the browser
runs the tool.
</Summary>

This is a stub. The full guide is authored in a later step.
